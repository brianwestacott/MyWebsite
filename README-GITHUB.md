## 🚀 Quick Deploy to GitHub Pages

1. **Create repo:** `BrianWestacott.github.io` or `phase1-mis2201`
2. `git init`
3. `git add .`
4. `git commit -m "Initial portfolio"`
5. `git remote add origin https://github.com/BrianWestacott/[repo].git`
6. `git push -u origin main`
7. **Settings → Pages → Source: Deploy from branch `main` `/ (root)`**
